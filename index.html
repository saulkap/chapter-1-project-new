Overview

FL studio is an application made for music production.

What can you do in FL studio?

In FL studio you can do anything from making just a beat to pluging in instruments to importing vocals and putting a song together.

What comes with FL studio?

FL studio has a number of stock plugins like FL keys which allows you to use a number of different pianos. You can also download drum kits, and other plugins to improve your production.

Terms Meanings
Drum Kit A set of Drum sounds from hats to kicks
Plugin Tools used to create sound or improve existing sounds
